解决Debian系统vi编辑器方向键为ABCD 退格键失效

itadmin | 2021年10月18日

最近使用Debian系统，需要修改一些文件，但是进入VI编辑器后选择插入模式，不能按上下左右和退格键很难受。

方法一，卸载vi编辑器改成vim编辑器

方法二，修改vi编辑器配置文件修复这个问题

首先vi /etc/vim/vimrc.tiny

然后小心翼翼的找到set compatible

移动到c字母上，进入插入模式（I）打上no两个字，然后ESC退出插入模式

再次移动到下一行，复制set backspace=2，插入进去。

最终代码为

set nocompatible 
set backspace=2

好了，这个问题就修复了
